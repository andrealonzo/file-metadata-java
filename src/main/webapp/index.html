File Metadata Microservice

User stories:

1) I can submit a FormData object that includes a file upload.

2) When I submit something, I will receive the file size in bytes within the JSON response

Example API implementation:

Example file metadata usage:

Post a Form Data object to the endpoint below. The name of the file input field must be "file".
/fileanalyze/

Example file metadata output:

{"fileSize":"2084","encoding":"7bit","mimetype":"image/gif","filename":"myfile.gif"}
